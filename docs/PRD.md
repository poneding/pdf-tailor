# PDF裁缝 - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位

PDF裁缝，支持PDF文档的切分和合并功能。完全在浏览器端处理，无需上传文件到服务器，保护用户隐私。

### 1.2 目标用户

- 需要快速处理PDF文档的办公人员
- 对隐私有要求、不愿上传文件到第三方服务器的用户
- 需要批量处理PDF的用户

### 1.3 核心价值

- **隐私安全**：纯浏览器端处理，文件不离开本地
- **简单易用**：无需安装软件，打开网页即可使用
- **免费开源**：无广告、无水印、无使用限制

---

## 2. 功能需求

### 2.1 PDF切分功能 (核心)

#### 2.1.1 按页数切分

| 功能点 | 描述 | 优先级 |
| ------ | ---- | ------ |
| 固定页数切分 | 每N页切分为一个文件（如每5页一个PDF） | P0 |
| 自定义页码范围 | 指定切分范围，如"1-3, 5, 8-10" | P0 |
| 单页提取 | 将每一页单独导出为独立PDF | P1 |
| 预览功能 | 切分前预览每个切分块包含的页面 | P1 |

#### 2.1.2 切分模式

- **均分模式**：将PDF平均切分为N份
- **按页数模式**：每X页切分一次
- **自定义模式**：手动指定页码范围

### 2.2 PDF合并功能

| 功能点 | 描述 | 优先级 |
| ------ | ---- | ------ |
| 多文件合并 | 将多个PDF合并为一个 | P0 |
| 拖拽排序 | 拖拽调整PDF合并顺序 | P0 |
| 选择性合并 | 从每个PDF选择特定页面进行合并 | P1 |
| 批量添加 | 一次性添加多个PDF文件 | P0 |

### 2.3 辅助功能

| 功能点 | 描述 | 优先级 |
| ------ | ---- | ------ |
| PDF预览 | 缩略图预览PDF各页面 | P1 |
| 页面旋转 | 旋转指定页面 | P2 |
| 页面删除 | 删除不需要的页面 | P2 |
| 批量下载 | 切分后的文件打包为ZIP下载 | P0 |

## 3. 用户体验需求

### 3.1 交互设计

#### 文件上传

- 支持拖拽上传
- 支持点击选择文件
- 支持批量选择多个文件
- 显示上传进度

#### 操作流程

```txt
上传PDF → 预览/选择页面 → 设置切分/合并参数 → 预览结果 → 下载
```

### 3.2 界面要求

- 响应式设计，支持桌面端和移动端
- 简洁直观的操作界面
- 实时显示处理进度
- 友好的错误提示

### 3.3 性能要求

| 指标 | 要求 |
| ---- | ---- |
| 文件大小限制 | 支持至少100MB的PDF |
| 处理速度 | 100页PDF处理时间 < 10秒 |
| 首屏加载 | < 3秒 |

---

## 4. 非功能需求

### 4.1 安全性

- 所有处理在浏览器端完成
- 不收集用户上传的任何文件内容
- 处理完成后自动释放内存

### 4.2 兼容性

- 支持主流现代浏览器（Chrome, Firefox, Safari, Edge）
- 支持最新两个主版本

### 4.3 可访问性

- 支持键盘操作
- 提供适当的ARIA标签
